# ModIA – Modeling and Artificial Intelligence  
Double Engineering Degree – INSA Toulouse × ENSEEIHT

This repository gathers the lectures, tutorials (TDs), practical sessions (TPs), and academic projects completed during the **ModIA (Modeling and Artificial Intelligence)** double engineering degree program between INSA Toulouse and ENSEEIHT.

## About the program 
ModIA is an advanced engineering track focused on mathematical modeling and artificial intelligence, combining strong theoretical foundations with hands-on implementation in machine learning and scientific computing.
The program is jointly offered by:
- INSA Toulouse 
- INP-ENSEEIHT

## Main topics covered
- Machine Learning
- Deep Learning
- Optimization
- Probability & Statistics
- Scientific Computing
- Signal Processing
- Mathematical Modeling
- Big data & Cloud

Feel free to explore the folders and projects!
